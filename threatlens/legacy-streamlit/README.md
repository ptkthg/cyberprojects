# ThreatLens Legacy Streamlit (MVP)

A versão MVP em Streamlit foi preservada no diretório raiz `threatlens/` para migração incremental segura.

## Como executar o legado

```bash
cd threatlens
streamlit run app.py
```

> Esta pasta marca o namespace de migração para a versão legado e a futura movimentação completa do código Streamlit.
